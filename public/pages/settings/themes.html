Custom Image

Change the image that is shown in the top left of the screen.

The image url MUST start with `https://`.

Notes

If you'd like to set the custom image to one used in a theme, use the name of the theme in all lowercase. Example: If you chose the theme 'Green' you would paste in 'green' into the prompt. Use 'logo' for the default icon.
